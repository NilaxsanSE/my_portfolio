# Suvendralingam Nilaxsan Portfolio

Personal portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Vercel.

Live site:

https://nilaxsan-dev-portfolio.vercel.app/

## Development

```bash
npm run dev
```

Open http://127.0.0.1:3000.

## Checks

```bash
npm run lint
npm run build
```

## Deployment

The site is connected to Vercel. Pushing to the `main` branch triggers a new deployment.
